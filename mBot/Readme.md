This is a project with mBot from Makeblock a company in china.
More infos are aviable here = https://de.wikipedia.org/wiki/MBot

This project will use the mBlock basic kit.
The robot is by default in the pause mode when button B in the IR remote is pressed it will start moving , if any obsticales are detected within 14cm from the ultrasonis sensor it will rotate (to the left)
until it finds a clear path then countinues
